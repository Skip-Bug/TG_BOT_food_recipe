# Сервис FoodPlan

Telegram-бот с рецептами по подписке.  
Имеется разделение доступа для пользователей без подписки, подписчиков и администраторов.


## Функционал

### Общий:
- **Рекомендация рецептов** - автоматический подбор рецепта
- **Список покупок** - автоматическая генерация списка ингредиентов
- **Контроль бюджета** - расчет стоимости блюд

### Для пользователя без подписки:
- **Общий**
- **Ограничение количества рецептов** - возможность просмотреть только до 3 рецептов
- **Оформление подписки** - возможность оплатить подписку и стать подписчиком с безлимитным доступом к рецептам

### Для подписчика:
- **Общий**
- **Ограничение количества рецептов** - без ограничений
- **Избранное** - сохранение понравившихся рецептов, удаление рецептов из избранного

### Для администратора:
- **Для подписчика**
- **Администрирование** -
    - возможность просмотреть список всех рецептов,
    - возможность удалить рецепты из списка,
    - возможность добавить новые рецепты.

## Запуск

Для запуска бота у вас должен быть установлен Python 3.

- Установите зависимости командой:
  ```
  pip install -r requirements.txt
  ```
- Определите настройки Telegram-бота в специальном файле ".env".  
Для этого создайте файл ".env" рядом с "bot.py" и запишите туда данные в формате: ПЕРЕМЕННАЯ=значение.  
    Потребуются следущие переменные:  
    - `TG_BOT_TOKEN` — API-токен Telegram-бота.
    - `ADMIN_ID` — числовой ID Telegram-пользователя, который будет администратором бота. Можно указать несколько ID через запятую.  
    
    Пример заполнения файла:
    ```
    TG_BOT_TOKEN=958423683:AAEAtJ5Lde5YYfkjergber
    ADMIN_ID=1069447500,10894230502
    ```
- Запустите бота командой:
  ```
  python bot.py
  ```
  Если настройки в файле ".env" введены верно, бот готов к работе.
